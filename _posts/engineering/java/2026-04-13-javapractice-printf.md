자바에서 데이터를 화면에 출력할 때 가장 많이 쓰는 것은 `System.out.println()`이다. 하지만 "이름은 김철수이고, 나이는 20살입니다."처럼 텍스트 중간중간에 변수를 끼워 넣으려면 `+` 기호를 계속 써야 해서 코드가 지저분해지고 오타가 나기 쉽다.

```java
// 🚨 기존 방식: + 기호 때문에 코드가 지저분하고 읽기 힘듦
System.out.println("이름은 " + name + "이고, 나이는 " + age + "살, 키는 " + height + "cm입니다.");
```

이럴 때 구멍(형식 지정자)을 뚫어놓은 문장 틀을 먼저 만들고, 그 구멍에 변수를 순서대로 쏙쏙 채워 넣는 **`System.out.printf()`**를 사용하면 코드가 훨씬 깔끔해진다. (`f`는 formatted의 약자다.)

## 1. `printf` 핵심 형식 지정자 (기본)

가장 자주 쓰이는 4가지 기본 기호는 다음과 같다. 기호 앞에는 항상 퍼센트(%)를 붙여서 사용한다.

기호 | 설명 | 사용 예시 | 실행 결과
:--|:--|:--|:--
`%d` | 정수 (Decimal) | `System.out.printf("%d", 1);` | 1
`%f` | 실수 (Float) | `System.out.printf("%f", 1.2);` | 1.200000
`%s` | 문자열 (String) | `System.out.printf("%s", "철수");` | 철수
`%n` | 줄바꿈 (Newline)| `System.out.printf("%s%n", "철수");` | 철수 (다음 줄로 넘어감)

**💡 참고 (`%n`과 `\n`)**
자바에서 줄바꿈을 할 때 `\n`을 써도 동작하지만, 운영체제(윈도우, 맥)에 상관없이 가장 안전하게 줄바꿈을 보장하는 표준 기호는 `%n`이다. `printf` 안에서는 `%n`을 쓰는 것을 습관화하자.

## 2. `printf` 여백과 정렬 제어 (고급)

단순히 변수를 넣는 것을 넘어, 콘솔 창에 표(Table)나 영수증처럼 줄을 딱딱 맞춰서 출력해야 할 때 사용하는 기호들이다. `%`와 영문자(`d`, `f`, `s`) 사이에 넣어서 사용한다.

기호 | 설명 | 사용 예시 | 실행 결과 (빈칸은 _로 임의 표시)
:--|:--|:--|:--
숫자 | 전체 칸 수 지정 (기본: 우측 정렬) | `printf("%4d", 1);` | ___1 (4칸을 확보하고 오른쪽에 붙임)
`-` | 왼쪽 정렬 | `printf("%-4d", 1);` | 1___ (4칸을 확보하고 왼쪽에 붙임)
`+` | (양수/음수) 부호 표시 | `printf("%+4d", 1);` | __+1
`0` | 빈 공간을 0으로 채우기 | `printf("%04d", 1);` | 0001 (4칸 중 빈칸을 0으로 채움)
`,` | 세 자리마다 콤마(,) 구분 | `printf("%,d", 1000);` | 1,000 (금액 표기에 유용)
`.` | 소수점 자리 지정 | `printf("%.2f", 1.234);` | 1.23 (소수점 셋째 자리에서 반올림)

## 3. 💻 코드로 보는 printf 실전 활용

위의 기호들을 조합하여 깔끔한 결제 영수증을 출력해 보자.

```Java
public class PrintfEx {
    public static void main(String[] args) {
        
        String menu = "아메리카노";
        int price = 4500;
        int count = 2;
        double taxRate = 0.1; // 10%

        System.out.println("====== 영수증 ======");
        
        // 1. %s, %d 조합 (단순 출력)
        System.out.printf("상품명: %s%n", menu);
        
        // 2. - 기호로 왼쪽 정렬, 숫자 지정으로 간격 맞추기
        System.out.printf("%-6s : %4d잔%n", "주문수량", count); // [주문수량_ : ___2잔]
        
        // 3. , 기호로 금액에 콤마 찍기
        System.out.printf("%-6s : %,d원%n", "결제금액", price * count); // [결제금액_ : 9,000원]
        
        // 4. . 기호로 소수점 제어, + 기호로 부호 표시
        // %%를 입력하면 실제 % 기호가 출력된다.
        System.out.printf("%-6s : %+.1f%%%n", "부가세율", taxRate * 100); // [부가세율_ : +10.0%]
        
        // 5. 0 기호로 빈자리 0 채우기 (영수증 번호 등)
        System.out.printf("영수증 번호: %05d%n", 12); // [00012]
    }
}
```
