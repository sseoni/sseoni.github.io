## 1. 리스트(List) 만들기

리스트는 대괄호 `[]`를 사용하여 만든다. 숫자, 문자열뿐만 아니라 또 다른 리스트까지 파이썬의 모든 자료형을 섞어서 담을 수 있는 유연한 바구니이다.

```python
# 빈 리스트 만들기
empty_list = []

# 다양한 데이터를 담은 리스트
student = ["홍길동", 20, True, [100, 90, 80]]
```


## 2. 데이터 꺼내기 (인덱싱과 슬라이싱)

문자열과 마찬가지로 0부터 시작하는 인덱스 번호를 사용해 원하는 데이터를 꺼내거나 잘라낼 수 있다.

```Python
colors = ["빨강", "파랑", "노랑", "초록"]

print(colors[0])   # 첫 번째 데이터: '빨강'
print(colors[-1])  # 뒤에서 첫 번째 데이터: '초록'
print(colors[1:3]) # 1번부터 2번까지 자르기: ['파랑', '노랑']
```


## 3. 리스트 데이터 조작하기 (메소드 사용)

리스트는 데이터의 추가, 수정, 삭제가 자유롭다. 변수 이름 뒤에 `.`을 찍어 다양한 기능(메소드)을 사용할 수 있다.

| 기능 | 메소드/문법 | 설명 |
|:--|:--|:--|
| 추가 | `.append(값)` | 리스트 맨 마지막에 데이터를 하나 추가한다. |
| 삽입 | `.insert(위치, 값)` | 원하는 인덱스 위치에 데이터를 끼워 넣는다. |
| 수정 | `리스트[인덱스] = 값` | 특정 위치의 데이터를 새로운 값으로 덮어쓴다. |
| 삭제 | `.remove(값)` | 리스트에서 첫 번째로 나오는 해당 값을 지운다. |
| 꺼내기 | `.pop()` | 맨 마지막 데이터를 리스트에서 빼내고(삭제) 그 값을 반환한다. |
| 정렬 | `.sort()` | 데이터를 가나다순이나 오름차순으로 예쁘게 정렬한다. |

```Python
# [리스트 메소드 활용 예제]
cart = ["사과", "바나나"]

cart.append("수박")       # 맨 뒤에 수박 추가 -> ['사과', '바나나', '수박']
cart.insert(1, "포도")    # 1번 자리에 포도 끼워넣기 -> ['사과', '포도', '바나나', '수박']
cart.remove("사과")       # 사과 삭제 -> ['포도', '바나나', '수박']

last_item = cart.pop()    # 맨 마지막 수박을 꺼냄
print(last_item)          # 출력: 수박
print(cart)               # 출력: ['포도', '바나나'] (수박이 빠진 상태)
```
