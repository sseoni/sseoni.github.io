프로그래밍을 시작할 때 가장 먼저 배우는 함수가 `print()`입니다.

하지만 코딩 테스트나 실기 시험장에서는 이 만만해 보이는 `print()`의 **출력 형식(공백, 줄바꿈, 괄호 등)** 하나 때문에 정답과 오답이 갈리곤 합니다.

시험에서 헷갈리지 않도록 `print()` 함수의 모든 것을 정리했습니다.


## 1. print()의 숨겨진 기본 옵션: `sep`과 `end`

`print()` 함수로 여러 개의 값을 출력할 때, 파이썬은 기본적으로 값들 사이에 '공백'을 넣고, 출력이 끝나면 '줄바꿈'을 합니다. 이 규칙을 마음대로 조작할 수 있는 옵션이 바로 `sep`과 `end`입니다.

```python
# 기본 출력: 띄어쓰기로 구분되고 끝나면 줄이 바뀜
print("A", "B", "C")  
# 출력: A B C

# 1. sep (Separator): 값들 사이에 넣을 문자 지정
print("A", "B", "C", sep="-") 
# 출력: A-B-C

print("A", "B", "C", sep="") # 공백 없이 붙여서 출력
# 출력: ABC

# 2. end: 출력이 다 끝난 후 맨 마지막에 넣을 문자 지정 (기본값은 \n 줄바꿈)
print("Hello", end=" ")
print("World")
# 출력: Hello World (두 줄로 나오지 않고 한 줄로 이어짐)
```

## 2. 문자열 포맷팅 (출력물 예쁘게 빚기)

변수와 문자를 섞어서 출력해야 할 때, 콤마(`,`)나 더하기(`+`)로 연결하면 코드가 지저분해집니다. 현재 파이썬에서 가장 많이 쓰이는 `f-string`을 무조건 익혀두세요.

```Python
name = "파이썬"
age = 30

# ❌ 과거의 방식 (C언어 스타일, 복잡함)
print("이름: %s, 나이: %d" % (name, age))

# 🔺 과도기 방식 (.format 메서드)
print("이름: {}, 나이: {}".format(name, age))

# ✅ 현재의 정답 (f-string): 문자열 앞에 'f'만 붙이고 변수를 중괄호 {} 안에 쏙!
print(f"이름: {name}, 나이: {age}")
# 출력: 이름: 파이썬, 나이: 30

# f-string의 사기적인 기능: 안에서 연산도 가능합니다.
print(f"내년 나이는 {age + 1}살 입니다.")
```

## 3. 🚨 0점 방지용: 컬렉션 자료형 출력과 함정

리스트, 튜플, 세트, 딕셔너리를 출력하면 각자의 고유한 괄호 기호가 함께 출력됩니다. 이 출력 형태와 '비어있을 때' 혹은 '요소가 하나일 때'의 함정을 반드시 기억하세요.

| 자료형 | 기호 | 출력 예시 | 🚨 주의해야 할 함정 출력 |
| -- | -- | -- | -- |
| 리스트(List) | [] | [1, 2, 3] | [] (빈 리스트) |
| 튜플(Tuple) | () | (1, 2, 3) | (1,) (요소가 1개일 땐 반드시 콤마가 붙음!) |
| 딕셔너리(Dict) | {} | {'A': 1, 'B': 2} | {} (빈 딕셔너리) |
| 세트(Set) | {} | {1, 2, 3} | set() (비어있을 땐 {}가 아니라 set()으로 출력됨!) |

### 💡 시험 빈출 함정 코드

```Python
# 함정 1: 요소가 하나인 튜플
print((5))    # 출력: 5 (튜플이 아니라 그냥 숫자 5로 인식함)
print((5,))   # 출력: (5,) (반드시 콤마가 있어야 튜플로 인정!)

# 함정 2: 빈 딕셔너리와 빈 세트
empty_box = {}
print(empty_box) # 출력: {} (파이썬은 빈 {}를 무조건 딕셔너리로 취급함)

empty_set = set()
print(empty_set) # 출력: set() (그래서 빈 세트는 이렇게 출력됨)
```

## 📌 실전 치트키: 리스트 괄호 벗기고 출력하기

정답이 `1 2 3` 인데, 리스트를 그냥 `print(arr)` 해버리면 `[1, 2, 3]`으로 출력되어 틀립니다. 이때는 언패킹 연산자(`*`)를 사용하세요.

```Python
arr = [1, 2, 3, 4, 5]

print(arr)   # 출력: [1, 2, 3, 4, 5] ❌
print(*arr)  # 출력: 1 2 3 4 5 ✅ (리스트의 껍데기를 까고 알맹이만 던짐)
```
