파이썬에서는 텍스트 데이터를 표현할 때 **문자열(`str`)**을 사용한다.

자바의 `char`와 `String` 구분 없이, 한 글자든 여러 글자든 모두 따옴표로 감싸면 문자열 객체가 된다.

**❗️주의**: 파이썬에서는 숫자 `123`은 숫자형이지만, 따옴표로 감싼 `"123"`은 문자열로 인식되어 다른 타입으로 취급된다.

## 1. 문자열 (String) 만들기

| 생성 방법 | 기호 | 특징 | 예시 |
| --- | --- | --- | --- |
| **큰따옴표** | `""` | 가장 일반적인 형태 | `"Hello"` |
| **작은따옴표** | `''` | 큰따옴표와 동일하게 동작 | `'Python'` |
| **따옴표 3개** | `""" """`<br>`''' '''` | 여러 줄의 문장을 줄바꿈 그대로 저장할 때 유용 | `"""Life is short,<br>You need Python"""` |

```python
"Hello World" # 큰따옴표
'Python is fun' # 작은따옴표

"""Life is too short, You need python"""    # 큰따옴표 3개

'''
Life is too short,
You need python
'''    # 작은따옴표 3개
```

**💡 Tip 여러 줄 문자열 처리하기**
줄바꿈을 표현할 때는 위의 예시처럼 큰따옴표/작은따옴표 3개를 연속으로 사용해 표현할 수도 있지만,
이스케이프 코드 `\n`을 사용하는 방법도 있다.


## 2. 탈출 문자 (Escape Code)

문자열 안에서 줄바꿈을 하거나, 문자열을 만드는 기호인 따옴표 자체를 글자로 출력하고 싶을 때 사용하는 특수 기호다. 역슬래시(`\`)와 문자를 결합하여 사용한다.

코드 | 설명 | 코드 | 설명
:--|:--|:--|:--
`\n` | 개행 (줄바꿈) | `\b` | 백 스페이스
`\v` | 수직 탭 | `\000` | 널(Null) 문자
`\t` | 수평 탭 | `\\` | 문자 \ (역슬래시) 자체
`\r` | 캐리지 리턴 | `\'` | 단일 인용부호 '
`\f` | 폼 피드 | `\"` | 이중 인용부호 "
`\a`| 벨 소리 | |

**💡 탈출 문자 없이 따옴표 포함하기**
파이썬은 작은따옴표와 큰따옴표를 모두 지원하므로, 문자열 안에 큰따옴표를 쓰고 싶다면 겉을 작은따옴표로 감싸면 탈출 문자 없이도 깔끔하게 작성할 수 있다.

```Python
# [문자열 생성 및 탈출 문자 예제]
# 1. 겉을 큰따옴표로 감싸서 안의 작은따옴표를 문자로 인식하게 함
sentence = "Python's favorite food is pearl."

# 2. 탈출 문자를 사용하여 줄바꿈 처리
greeting = "안녕하세요.\n반갑습니다."
```


## 3. 문자열 연산

파이썬만의 강력한 특징 중 하나는 숫자처럼 문자열도 더하거나 곱할 수 있다는 점이다.

더하기 (`+`): 두 문자열을 이어 붙인다.
곱하기 (`*`): 문자열을 주어진 숫자만큼 반복한다.

```Python
# [문자열 더하기와 곱하기]
head = "Hello"
tail = " Python!"

# 문자열 더하기 (이어 붙이기)
print(head + tail) 
# 출력: Hello Python!

# 문자열 곱하기 (반복하기)
line = "-"
print(line * 20)
# 출력: -------------------- (선 긋기 효과 등에 유용함)
```
