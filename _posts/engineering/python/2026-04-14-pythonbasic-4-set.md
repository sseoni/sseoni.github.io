## 1. 세트(Set)의 두 가지 특징

세트는 중괄호 `{}`를 사용하거나 `set()` 함수를 통해 만든다. 중고등학교 수학 시간에 배운 '집합'과 완벽히 동일한 특징을 가진다.

1. **중복을 절대 허용하지 않는다.** (같은 값을 여러 개 넣어도 하나만 남는다.)
2. **순서가 없다.** (인덱스 번호가 없으므로 `set[0]`처럼 위치로 값을 꺼낼 수 없다.)

```python
# 세트 만들기 (중복된 2와 3이 자동으로 하나씩만 남음)
my_set = {1, 2, 2, 3, 3, 3}
print(my_set) # 출력: {1, 2, 3}

# 문자열을 세트로 만들면 알파벳이 쪼개지고 중복이 제거됨
hello_set = set("Hello")
print(hello_set) # 출력: {'H', 'e', 'l', 'o'} (순서는 무작위로 출력됨)

# 순서가 없기 때문에 위치로 접근 불가
print(hello_set[0]) # ❗️ 에러 발생
```


## 2. 수학보다 쉬운 집합 연산

데이터 그룹 간의 관계를 분석할 때 세트의 집합 연산은 강력한 무기가 된다.

```Python
group_A = {"유재석", "박명수", "정준하"}
group_B = {"유재석", "노홍철", "하하"}

# 1. 교집합 (& 기호 또는 .intersection() 사용)
print(group_A & group_B) # 두 그룹에 모두 있는 사람: {'유재석'}

# 2. 합집합 (| 기호 또는 .union() 사용)
print(group_A | group_B) # 두 그룹의 모든 사람 (중복은 1명으로 처리)

# 3. 차집합 (- 기호 또는 .difference() 사용)
print(group_A - group_B) # 그룹 A에만 있는 사람: {'박명수', '정준하'}
```


## 3. 데이터 추가와 삭제

순서가 없기 때문에 특정 위치에 밀어 넣는 `insert()`는 없고, 그냥 값을 던져 넣는 `add()`를 사용한다.

`.add(값)`: 값 1개 추가

`.update([값1, 값2])`: 여러 개의 값을 한 번에 추가

`.remove(값)`: 특정 값 삭제 (값이 없으면 에러 발생)
