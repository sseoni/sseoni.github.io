프로그래밍을 하다 보면 숫자처럼 생겼지만 문자열로 취급되거나, 실수를 정수로 바꿔서 계산해야 하는 등 데이터의 형태를 바꿔야 하는 상황이 빈번하게 발생한다.

이때 사용하는 개념이 바로 형변환(Type Conversion), 또는 **캐스팅(Casting)**이다.

## 1. 형변환이란? (자동 형변환 vs 수동 형변환)

파이썬의 형변환은 크게 두 가지로 나뉜다.

### 암시적 형변환 (Implicit Type Conversion)

- 파이썬이 알아서 데이터 타입을 변환해 주는 기능
- 주로 데이터 손실을 막기 위해 더 넓은 범위의 타입으로 자동 변환된다.

### 명시적 형변환 (Explicit Type Conversion)

- 개발자가 직접 내장 함수를 사용해 데이터 타입을 강제로 변환하는 것

## 암시적 형변환: 파이썬 자동 변환

파이썬 내부 엔진이 데이터의 손실(소수점 이하 버림 등)을 방지하기 위해 좁은 범위의 정수를 더 넓은 범위의 실수로 자동 변환하여 연산한다.

```Python
# 암시적 형변환 예시
num_int = 3      # 정수형
num_float = 1.2  # 실수형

# 정수와 실수를 더하면 결과는 자동으로 실수형이 된다.
result = num_int + num_float 

print(result)       # 출력: 4.2
print(type(result)) # 출력: <class 'float'>
```

## 2. 명시적 형변환: 내장 함수 활용하기

개발자가 직접 타입을 바꿔야 할 때는 파이썬의 내장 함수를 사용한다. 첨부해 주신 이미지의 내용이 바로 이 명시적 형변환의 핵심적인 예시들이다.

### 정수로 변환 int()

문자열이나 실수를 정수(int) 데이터로 변환한다. 실수형을 정수로 변환할 때는 소수점 이하를 버린다.

```Python
# 1. 문자열을 정수로 변환
str_num = '2'
converted_int = int(str_num) 
# 문자열 '2'가 정수 2로 안전하게 변환된다.

# 2. 실수를 정수로 변환
float_num = 3.8
converted_from_float = int(float_num) 
# 소수점 이하를 버리고 정수 3이 된다. (반올림 아님)
```

🚨 주의: 소수점이 있는 문자열을 바로 int()로 변환할 때
'2.5'와 같이 소수점을 포함한 문자열을 바로 int('2.5')로 변환하려고 하면 파이썬은 형태를 인식하지 못하고 에러(ValueError)를 발생시킵니다.
이럴 때는 실수로 먼저 변환한 뒤, 다시 정수로 변환해야 합니다.

Python
# 에러가 발생하는 경우를 방지하는 올바른 중첩 변환 방법
str_float = '2.5'

# int(str_float) -> ValueError 발생!

# 올바른 방법: 먼저 float()으로 감싸고, 그 결과를 int()로 감쌉니다.
safe_int = int(float(str_float))
print(safe_int) # 2.5가 실수로 변환된 후, 소수점이 버려져 2가 출력됩니다.
코드 설명: 문자열 형태의 실수를 정수로 캐스팅하기 위해 float() 함수로 먼저 파싱한 후, int() 함수를 통해 정수형으로 최종 변환하는 로직입니다.

2.2. 실수로 변환 float()

문자열이나 정수를 실수(float) 데이터로 변환합니다. 정수를 실수로 변환하면 끝에 .0이 붙습니다.

Python
# 1. 문자열을 실수로 변환
str_num = '1.5'
converted_float = float(str_num) # 1.5

# 2. 정수를 실수로 변환
integer_num = 5
converted_from_int = float(integer_num) # 5.0
코드 설명: 정수 데이터를 float()으로 변환하면 메모리 상에서 부동소수점 형태로 저장되며, 값의 손실 없이 .0이 추가된 형태로 표현됩니다.

2.3. 문자열로 변환 str()

숫자(정수, 실수)나 다른 데이터 형태를 화면에 출력하거나 글자와 합치기 위해 문자열(string)로 변환합니다.

Python
# 숫자를 문자열로 변환
age = 25
# print("나는 " + age + "살 입니다.") -> TypeError 발생 (문자열과 숫자는 더할 수 없음)

# 숫자를 str()을 이용해 문자열로 형변환하여 연결합니다.
message = "나는 " + str(age) + "살 입니다."
print(message) # 출력: 나는 25살 입니다.
코드 설명: 서로 다른 데이터 타입 간의 연결(Concatenation) 연산을 수행하기 위해 정수형 변수를 str() 함수를 사용하여 문자열 객체로 변환합니다.

2.4. 논리형으로 변환 bool()

어떤 값을 참(True)이나 거짓(False)으로 변환합니다. 파이썬에서는 값이 비어있거나 0이면 False, 그 외의 값이 있으면 True로 판별합니다.

Python
# 거짓(False)으로 변환되는 경우 (비어있거나 0인 경우)
print(bool(0))     # 숫자 0 -> False
print(bool(""))    # 빈 문자열 -> False
print(bool([]))    # 빈 리스트 -> False

# 참(True)으로 변환되는 경우 (값이 존재하는 경우)
print(bool(1))     # 0이 아닌 숫자 -> True
print(bool("안녕")) # 내용이 있는 문자열 -> True
3. 컬렉션 자료형 간의 변환 (리스트, 튜플, 세트)
숫자나 문자열뿐만 아니라, 데이터를 담는 바구니 역할을 하는 자료구조들(리스트, 튜플, 세트) 사이에서도 자유롭게 형변환이 가능합니다. 이 방법은 데이터의 중복을 제거하거나, 데이터를 수정하지 못하게 잠글 때 매우 유용합니다.

Python
# 리스트를 세트(집합)로 변환하여 중복 제거하기
my_list = [1, 2, 2, 3, 3, 3]
my_set = set(my_list) 
# 중복이 허용되지 않는 세트의 특성 덕분에 {1, 2, 3}만 남습니다.

# 중복이 제거된 세트를 다시 리스트로 변환하기
clean_list = list(my_set)
print(clean_list) # 출력: [1, 2, 3]
코드 설명: 컬렉션 타입 캐스팅을 활용한 데이터 전처리 기법입니다. 리스트를 set()으로 캐스팅하여 고유값만 추출한 뒤, 순서가 있는 배열로 다시 사용하기 위해 list()로 재캐스팅합니다.
