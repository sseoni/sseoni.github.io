`" "`따옴표로 둘러싸서 표현한다.
파이썬에서는 숫자 `123`은 숫자형이지만, 따옴표로 감싼 `"123"`은 문자열로 인식되어 다른 타입으로 취급된다.

## 1. 문자열 생성 방법
큰따옴표와 작은따옴표로 문자열을 생성할 수 있다.

```python
"Hello World" # 큰따옴표
'Python is fun' # 작은따옴표

"""Life is too short, You need python"""    # 큰따옴표 3개

'''
Life is too short,
You need python
'''    # 작은따옴표 3개
```

💡 팁: 여러 줄 문자열 처리하기
줄바꿈을 표현할 때는 이스케이프 코드 \n을 삽입하거나, 큰따옴표/작은따옴표 3개를 연속으로 사용하면 편하게 여러 줄을 작성할 수 있다.

## 2. 이스케이프 코드 (Escape Code)

문자열 안에서 특수한 기능을 수행하도록 약속된 문자 조합이다.

코드 | 설명 | 코드 | 설명
:--|:--|:--|:--
`\n` | 개행 (줄바꿈) | `\b` | 백 스페이스
`\v` | 수직 탭 | `\000` | 널(Null) 문자
`\t` | 수평 탭 | `\\` | 문자 \ (역슬래시) 자체
`\r` | 캐리지 리턴 | `\'` | 단일 인용부호 '
`\f` | 폼 피드 | `\"` | 이중 인용부호 "
`\a`| 벨 소리 | |

## 3. 문자열 연산

파이썬에서는 문자열도 더하거나 곱할 수 있다.

더하기 (`+`): 두 문자열을 이어 붙인다.

곱하기 (`*`): 문자열을 주어진 숫자만큼 반복한다.
