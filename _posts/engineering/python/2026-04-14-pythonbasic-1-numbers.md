## 1. 숫자형 (Numbers)

파이썬에서 숫자를 다루는 자료형은 크게 정수형과 실수형으로 나뉜다. 복잡한 타입 선언 없이 숫자를 변수에 넣기만 하면 파이썬이 알아서 타입을 결정한다.

| 분류 | 자료형 | 설명 | 예시 코드 |
| --- | --- | --- | --- |
| **정수형** | `int` | 소수점이 없는 모든 양수, 음수, 0 | `int age = 25;` (x)<br>`age = 25` (o) |
| **실수형** | `float` | 소수점이 포함된 숫자 | `pi = 3.14` |
| **복소수** | `complex` | 실수부와 허수부(`j`)로 구성된 수 | `c = 1 + 2j` |

### 정수 (int)
소수점이 없는 양수, 음수, 0
8진수와 16진수도 정수에 포함된다
    - **8진수**: 숫자 0과 알파벳 o(또는 O)로 시작
    - **16진수**: 숫자 0과 알파벳 x(또는 X)로 시작

### 실수 (float)
소수점이 포함된 숫자
지수 표기법으로도 표현가능

**지수 표기법 (E, e)**
    ```4.24E10```은 $4.24 \times 10^{10}$을 의미하고, ```4.24e-10```은 $4.24 \times 10^{-10}$을 의미한다.

**복소수 (complex)**
    실수부와 허수부로 구성되며, 허수 뒤에 j를 붙여 표현한다.

``` Python
# 정수형 및 실수형 할당
a = 1.2
b = -3.45

# 지수 표기법
c = 4.24E10

# 복소수 생성 및 내장 속성/함수 활용
d = 1 + 2j
print(d.real)        # 복소수의 실수부를 반환
print(d.imag)        # 복소수의 허수부를 반환
print(d.conjugate()) # 켤레복소수를 반환
print(abs(d))        # 복소수의 절댓값을 반환
```

## 2. 숫자의 연산 (Operators)

파이썬은 일반적인 사칙연산 외에도 프로그래밍에 유용한 몫과 나머지 연산자를 직관적으로 제공한다.

**기본 사칙연산**: `+` (더하기), `-` (빼기), `*` (곱하기), `/` (나누기)
**특수 연산자**:
    `**` : 거듭제곱 (x의 y승)
    `//` : 몫 (나눗셈 후 소수점을 버린 정수 결과)
    `%` : 나머지 (나눗셈 후 남은 값, 짝수/홀수 판별에 자주 쓰임)

### 💡 파이썬의 나누기 연산(`/`) 주의점

파이썬 3.x 버전부터는 정수끼리 나누더라도 결과가 나누어떨어지지 않으면 자동으로 **실수형(`float`)**으로 결과를 반환한다. 

```Python
# [숫자형 연산 기본 예제]
int bread = 10
int people = 3

# 1. 일반 나누기 (결과가 자동으로 실수형으로 변환됨)
print(bread / people) 
# 출력: 3.3333333333333335

# 2. 몫만 구하기 (// 연산자)
print(bread // people)
# 출력: 3 (1명당 3개씩 가질 수 있음)

# 3. 나머지만 구하기 (% 연산자)
print(bread % people)
# 출력: 1 (1개가 남음)
```
