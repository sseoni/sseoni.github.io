## 1. 인덱스와 슬라이싱 (Indexing & Slicing)

문자열은 여러 글자가 순서대로 나열된 데이터다. 파이썬은 이 순서(인덱스)를 통해 특정 위치의 글자를 콕 집어내거나 잘라낼 수 있다.
**❗️ 파이썬의 인덱스는 0부터 시작한다.**

**인덱싱(`[번호]`)**: 특정 위치의 글자 하나를 뽑아낸다. (음수를 쓰면 뒤에서부터 센다.)
**슬라이싱(`[시작:끝]`)**: 지정한 범위의 글자를 잘라낸다. **끝 번호의 글자는 포함되지 않음**에 주의해야 한다.

```python
# [인덱싱과 슬라이싱 예제]
word = "Python"
# P(0) y(1) t(2) h(3) o(4) n(5)

# 인덱싱
print(word[0])  # 첫 번째 글자: 'P'
print(word[-1]) # 뒤에서 첫 번째 글자: 'n'

# 슬라이싱
print(word[0:2]) # 0번부터 1번까지 (2번은 미포함): 'Py'
print(word[2:])  # 2번부터 끝까지: 'thon'
```


## 2. 문자열 포맷 (Formatting)

문자열 안에 변수나 수시로 변하는 값을 끼워 넣고 싶을 때 사용한다.
파이썬 3.6부터 도입된 `f-string`이 가장 직관적이고 많이 쓰인다.

문자열 앞에 `f`를 붙이고 중괄호 `{}` 안에 변수를 넣으면 된다.

```Python
# [f-string 포맷팅 예제]
name = "서현"
age = 25
# f를 붙이고 중괄호 안에 변수명을 직접 입력한다.
print(f"제 이름은 {name}이고, 나이는 {age}살입니다.")
# 출력: 제 이름은 서현이고, 나이는 25살입니다.
```


## 3. 문자열 메소드 (String Methods)

문자열 객체 자체에 내장된 유용한 함수들이다. 변수 이름 뒤에 점(`.`)을 찍고 사용한다.
원본 문자열은 불변(Immutable)이므로, 메소드 실행 결과는 새로운 문자열로 반환된다.

| 메소드 | 설명 | 예시 코드 |
|:--|:--|:--|
| `split()` | 특정 문자(기본값: 공백)를 기준으로 문자열을 쪼개어 리스트로 반환 | `"A B C".split() -> ['A', 'B', 'C']` |
| `replace()` | 특정 문자를 다른 문자로 교체 | `"Java".replace("J", "L") -> "Lava"` |
| `strip()` | 문자열 양쪽의 공백을 제거 | `" hi ".strip() -> "hi"` |
| `join()` | 리스트의 요소들을 특정 문자로 연결하여 하나의 문자열로 만듦 | `"-".join(['010', '1234']) -> "010-1234"` |
