## 1. Key로 찾고 Value로 답한다

딕셔너리는 중괄호 `{}` 안에 `키(Key): 값(Value)` 형태를 쉼표로 연결하여 만든다.
데이터가 수만 개가 있어도 인덱스를 처음부터 뒤지지 않고 **Key(이름표)를 통해 데이터를 한 번에 찾아내기 때문에 검색속도가 빠르다.**

```python
# 딕셔너리 만들기
user = {
    "name": "서현",
    "age": 25,
    "skills": ["Python", "Java"]
}

# Key를 통해 Value 꺼내기
print(user["name"]) # 출력: 서현
```


## 2. 데이터 추가, 수정, 삭제

딕셔너리는 새로운 이름표를 붙이면 추가가 되고, 기존에 있는 이름표를 쓰면 덮어쓰기(수정)가 된다.

```Python
# 1. 데이터 추가 및 수정
user["job"] = "개발자" # 새로운 Key이므로 맨 끝에 데이터 추가됨
user["age"] = 26       # 이미 있는 Key이므로 값이 25에서 26으로 수정됨

# 2. 데이터 삭제 (del 키워드 사용)
del user["skills"]     # skills 키와 그 안의 데이터가 모두 삭제됨
```


## 3. 에러를 피하는 안전한 조회: `get()`

딕셔너리에서 대괄호 `[]`를 사용해 없는 `Key`를 찾으려고 하면 `KeyError`가 발생하면서 프로그램이 비정상 종료된다.
이때 `.get()` 메소드를 사용하면 안전하게 코드를 짤 수 있다.

```Python
# print(user["email"])  -> 에러(KeyError) 발생 후 프로그램 터짐!

# .get()을 사용하면 에러 대신 None(없음)을 반환하며 프로그램이 계속 돌아감
print(user.get("email")) 

# 팁: 데이터가 없을 때 띄워줄 기본 문구를 설정할 수도 있다.
print(user.get("email", "이메일 정보가 등록되지 않았습니다."))
```


## 4. 딕셔너리 핵심 메소드 3가지

딕셔너리 안의 `Key`만, 혹은 `Value`만 따로 묶어서 봐야 할 때 유용한 기능들이다. 주로 반복문(`for`)과 함께 쓰인다.

`.keys()`: 딕셔너리의 `Key`들만 모아서 돌려준다.

`.values()`: 딕셔너리의 `Value`들만 모아서 돌려준다.

`.items()`: Key와 Value를 튜플 `(Key, Value)` 형태로 예쁘게 묶어서 돌려준다.
