최근 정보처리기사 실기 C언어 파트에서 가장 핫한 주제는 단연 **연결 리스트(Linked List)**입니다. 배열처럼 크기를 미리 정해두지 않고, 필요할 때마다 기차 객실처럼 서로의 꼬리를 물고 연결되는 유연한 자료구조입니다. 이를 구현하기 위해서는 **'자기 참조 구조체'** 문법을 이해해야 합니다.


## 1. 자기 참조 구조체란?

구조체 안에 **"나랑 완전히 똑같이 생긴 다른 구조체의 주소"**를 담을 수 있는 포인터 방을 만들어 둔 것을 의미합니다.

```c
struct Node {
    int v;                  // 1. 자신의 데이터(알맹이)를 담는 방
    struct Node* next;      // 2. 다음번 기차 객실(Node)의 주소를 가리키는 포인터
};
```

## 2. 실습 코드: 꼬리 물기와 순회(Traversal)

흩어져 있는 노드들을 주소 연산자(`&`)로 연결하고, 화살표(`->`)를 통해 끝까지 순회하는 흐름을 완벽히 외워야 합니다.

```C
#include <stdio.h>

struct Node {
    int v;                  
    struct Node* next;      
};

int main() {
    // 1. 흩어져 있는 노드 3개 생성 
    // 아직은 서로 연결되지 않았으므로 다음 주소(next)를 NULL(없음)로 비워둡니다.
    struct Node n1 = {1, NULL};
    struct Node n2 = {2, NULL};
    struct Node n3 = {3, NULL};

    // 2. 🚨 꼬리 물기 (연결 리스트 생성)
    // n1 다음에 n3, n3 다음에 n2가 오도록 기차를 연결합니다. (순서 주의: 1 -> 3 -> 2)
    n1.next = &n3;
    n3.next = &n2;

    // 3. 리스트 순회 준비 (맨 처음 머리인 n1의 주소부터 출발)
    struct Node* c = &n1;
    
    // 4. c가 NULL(기차의 끝)을 만날 때까지 무한 반복
    while (c != NULL) {
        // c는 포인터이므로 구조체 멤버(v)에 접근할 때는 무조건 화살표(->) 사용!
        printf("%d ", c->v); 
        
        // 🚨 가장 중요한 핵심 로직: 현재 위치를 다음 칸으로 점프!
        c = c->next; 
    }
    // 최종 출력 결과: 1 3 2

    return 0;
}
```

### 💡 출제자의 함정

구조체끼리 연결할 때는 무조건 주소값(`&`)을 넘겨줘야 합니다. `(n1.next = &n3;)`
순회할 때 다음 칸으로 이동하는 공식은 **`현재포인터 = 현재포인터->next;`** 입니다.
