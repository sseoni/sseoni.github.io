C언어를 시작할 때 가장 먼저 부딪히는 벽은 '엄격한 데이터 타입(정적 타입)'과 '문자열(String)의 부재'입니다. 파이썬이나 자바스크립트처럼 변수에 아무 데이터나 자유롭게 넣을 수 있는 언어와 달리, C언어는 변수를 선언할 때 **'방의 크기와 용도'**를 정확히 명시해야 합니다.

## C언어의 필수 데이터 타입 3가지

프로그래밍에서 가장 많이 쓰이는 3가지 기초 데이터 타입입니다.

| 데이터 타입 | 의미 | 크기 | 저장 가능한 값의 예시 |
| :--- | :--- | :--- | :--- |
| **`int`** | 정수 (Integer) | 4 Byte | `10`, `-5`, `0` (소수점 없음) |
| **`float` / `double`** | 실수 (Float) | 4 / 8 Byte | `3.14`, `-0.5` (소수점 있음) |
| **`char`** | 문자 (Character) | 1 Byte | `'A'`, `'7'`, `'\n'` (무조건 딱 1글자) |

🚨 C언어에는 여러 글자를 한 번에 담는 **`String` (문자열) 타입이 아예 존재하지 않습니다.** 오직 1바이트 크기의 단일 문자를 담는 `char`만 지원합니다.

---

## 문자열은 1차원 문자 배열 (`char []`)이다.

그렇다면 "Apple"과 같은 단어는 어떻게 저장할까요? 글자 1개짜리 방(`char`)을 여러 개 이어 붙인 **'1차원 배열'**을 만들어야 합니다.

```c
#include <stdio.h>

int main() {
    // 1. 단일 문자열은 1차원 char 배열로 만듭니다. (쌍따옴표 사용)
    // 메모리에는 ['R', 'e', 'd', '\0'] 형태로 총 4칸이 저장됩니다.
    char color[4] = "Red"; 

    // %s 서식 지정자는 '\0'(널 문자)를 만날 때까지 연속으로 글자를 출력합니다.
    printf("선택한 색상: %s\n", color);

    return 0;
}
```

### Null 문자 (`\0`)

C언어는 배열 안에서 단어가 어디서 끝나는지 스스로 알지 못합니다. 그래서 단어의 제일 마지막에 항상 눈에 보이지 않는 종료 신호인 `\0`(널 문자)를 몰래 추가합니다. `"Red"`라는 3글자를 저장하기 위해 최소 4칸의 배열 크기가 필요한 이유입니다.

## 단어 리스트 저장하기: 2차원 문자 배열

`"Red"`라는 단어 1개가 1차원 배열이라는 것을 이해했다면, 다음 단계로 넘어갈 수 있습니다.
`"Red", "Green", "Blue"` 처럼 여러 개의 단어를 하나의 변수에 묶어서 저장하려면 어떻게 해야 할까요? 1차원 배열들이 모여 집합을 이뤄야 하므로, 자연스럽게 '2차원 배열'을 사용해야 합니다.

```C
#include <stdio.h>

int main() {
    // 1. 여러 개의 문자열을 담는 2차원 배열 선언
    // 앞의 대괄호 [] : 저장할 단어의 개수 (자동으로 3칸이 할당됨)
    // 뒤의 대괄호 [10] : 각 단어가 들어갈 방의 최대 길이 (널 문자 포함 최대 10칸)
    char types[][10] = {"Red", "Green", "Blue"};

    // 2. 반복문을 통한 배열 순회 및 출력
    // types[i]는 i번째 단어의 시작 주소를 의미합니다.
    for(int i = 0; i < 3; i++) {
        printf("%d번째 단어: %s\n", i, types[i]);
    }

    return 0;
}
```

`char types[][10]` 구조를 해석하면, "Null문자를 포함해 최대 10글자까지 들어갈 수 있는 방을 여러 개(여기서는 3개) 준비해라"라는 뜻이 됩니다. "Red"처럼 짧은 단어가 들어가더라도 남는 공간은 여유 공간으로 비워두게 됩니다.
