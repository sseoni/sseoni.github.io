정보처리기사 실기 시험에서 코딩 문제는 배점이 가장 높습니다. 그 중에서도 출력 결과의 형식을 결정하는 **서식 지정자(Format Specifier)**는 작은 실수 하나로 부분 점수 없이 오답 처리될 수 있는 핵심 포인트입니다. 가장 자주 출제되는 핵심 지정자 5가지를 정리합니다.

## 1. 핵심 서식 지정자 요약

| 서식 지정자 | 의미 | 데이터 타입 | 매칭되는 값 |
| :--- | :--- | :--- | :--- |
| **`%c`** | Character (문자 1개) | `char` | 'A', '7', '!' 등 딱 한 글자 |
| **`%s`** | String (문자열) | `char *` 또는 배열명 | "APPLE", "HELLO" 등 글자 배열 (주소값 필요) |
| **`%d`** | Decimal (정수, 10진수) | `int`, `short` 등 | 10, -5, 0 등 소수점이 없는 숫자 |
| **`%o`** | Octal (8진수) | `int`, `unsigned` 등 | 숫자를 8진수 형태로 출력 |
| **`%x`** | Hexadecimal (16진수) | `int`, `unsigned` 등 | 숫자를 16진수 형태로 출력 (소문자) |

> **꿀팁:** 16진수를 대문자(A~F)로 출력하고 싶다면 대문자 **`%X`**를 사용하면 됩니다.

---

## 2. 기초 실습 코드

```c
#include <stdio.h>

int main() {
    char ch = 'A';             // 문자 변수 선언
    char str[] = "KOREA";      // 문자열(배열) 선언
    int num = 2026;            // 정수형 변수 선언

    // 1. %c는 딱 한 글자만 출력합니다.
    printf("1. 문자 출력: %c\n", ch);

    // 2. %s는 문자열의 시작 주소부터 널 문자('\0')를 만날 때까지 연속으로 출력합니다.
    printf("2. 문자열 출력: %s\n", str);

    // 3. %d는 정수를 10진수로 출력합니다.
    printf("3. 정수 출력: %d\n", num);

    return 0;
}
```

## 3. 심화 실습 코드 (8진수와 16진수 변환)

정처기 시험에서는 10진수 값을 주고 `%o`나 `%x`로 출력했을 때의 결과를 묻는 문제가 자주 출제됩니다.

```C
#include <stdio.h>

int main() {
    // 10진수 255를 변수에 저장
    int num = 255; 

    // %d: 10진수 그대로 출력
    printf("10진수: %d\n", num); // 결과: 255

    // %o: 8진수로 변환하여 출력 (255 = 8진수로 377)
    printf("8진수: %o\n", num);  // 결과: 377

    // %x: 16진수로 변환하여 출력 (255 = 16진수로 ff)
    printf("16진수(소문자): %x\n", num); // 결과: ff
    
    // %X: 16진수 대문자로 변환하여 출력
    printf("16진수(대문자): %X\n", num); // 결과: FF

    return 0;
}
```

## 4. 시험에서 유의사항
**대소문자 구분**: `%c`, `%s`, `%d`, `%o`는 모두 소문자로 작성해야 합니다.
- (`%x`는 16진수를 소문자 출력, `%X`는 대문자 출력)
**줄바꿈 기호(`\n`)**: 출력 결과에 `\n`이 포함되어 있다면 답안 작성 시 반드시 줄을 바꾸어 가며 디버깅 결과를 적어야 합니다.
