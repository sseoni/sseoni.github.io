포인터는 변수의 값이 아닌 **'메모리 주소'**를 저장하는 특별한 변수입니다.

## 포인터 변수 선언 (`*`)

포인터 변수를 만들 때는 타입 뒤나 변수명 앞에 `*`를 붙입니다. 이는 "이 변수는 일반 데이터가 아니라 메모리 주소를 담는 방이다"라는 의미입니다.

포인터 변수의 자료형은 가리키고자 하는 원본 변수의 자료형과 반드시 일치해야 합니다.
아래 예시에서 `p_int`에는 정수형 변수의 주소만을 담을 수 있습니다.

```c
#include <stdio.h>

int main() {
    int a = 10;
    char c = 'A';

    int *p_int;      // 정수형 변수의 주소를 담는 포인터 선언
    char *p_char;    // 문자형 변수의 주소를 담는 포인터 선언

    return 0;
}
```

## 주소 연산자 (`&`)

일반 변수 앞에 `&`를 붙이면, 해당 변수가 메모리의 어디에 저장되어있는지 있는지 '주소값'을 반환합니다.

포인터 변수에 값을 집어넣을 때 사용됩니다.

```c
#include <stdio.h>

int main() {
    int score = 100;
    
    // score 변수의 주소값을 포인터 변수 ptr에 할당
    int *ptr = &score; 

    // %p는 포인터(메모리 주소)를 출력하기 위한 서식 지정자입니다.
    printf("score 변수의 실제 메모리 주소: %p\n", &score);
    printf("ptr 변수가 저장한 주소: %p\n", ptr); // 위와 완벽히 동일한 주소 출력

    return 0;
}
```


## 간접 참조 연산자 (`*`)

이미 선언된 포인터 변수 앞에 `*`를 붙이면, 해당 주소로 찾아가 그 안에 들어있는 '알맹이(실제 값)'를 반환합니다.

포인터 변수가 가리키는 주소가 담고있는 값을 꺼내거나 다른 값으로 수정할 때 사용합니다.

```c
#include <stdio.h>

int main() {
    int num = 10;
    int *p = &num; 

    // 간접 참조 연산자(*)로 값 읽기
    printf("초기 num의 값: %d\n", num);    // 출력값: 10
    printf("초기 *p의 값: %d\n", *p);      // 출력값: 10

    // 간접 참조 연산자(*)로 원본 값 수정하기
    // p가 가리키는 주소(num의 위치)로 가서 알맹이를 20으로 바꿈
    *p = 20; 
    
    // num을 직접 수정하지 않았지만, *p를 통해 num이 20으로 변경됨
    printf("수정 후 *p의 값: %d\n", *p);   // 출력값: 20
    printf("수정 후 num의 값: %d\n", num); // 출력값: 20

    return 0;
}
```


```c
#include <stdio.h>

int main() {
    int num = 10;       // 일반 변수 선언 및 초기화
    int *p = &num;      // 포인터 변수 p 선언, num의 메모리 주소 할당

    printf("%d\n", num);        // 출력값: 10 (num의 값)
    printf("%p\n", &num);       // 출력값 예시: 0x7ffeefbff5a8 (num의 메모리 주소)

    printf("%p\n", p);          // 출력값 예시: 0x7ffeefbff5a8 (포인터 p가 가진 주소값, num의 주소와 동일)
    printf("%d\n", *p);         // 출력값: 10 (포인터 p가 가리키는 값, *p는 num과 동일)

    *p = 20;                    // 포인터가 가리키는 주소의 값을 20으로 변경
    printf("%d\n", num);        // 출력값: 20 (변경된 num의 값)


    p_int = &a;      // 주소값을 담을 포인터 방 p_int에는 주소값(&a)을 담아야한다.
    p_char = &a;     // 🚨 에러 발생: 정수형 변수의 주소를 문자형 포인터에 담을 수 없음!
    

    return 0;
}
```
