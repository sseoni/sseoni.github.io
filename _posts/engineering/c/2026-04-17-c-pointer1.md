// [Post 1 Code: 포인터 기본]
#include <stdio.h>

int main() {
    int num = 10;       // 일반 변수 선언 및 초기화
    int *p = &num;      // 포인터 변수 p 선언, num의 메모리 주소 할당

    printf("num의 값: %d\n", num);         // 10
    printf("num의 메모리 주소: %p\n", &num); // 예: 0x7ffeefbff5a8

    printf("포인터 p가 가진 주소값: %p\n", p); // 예: 0x7ffeefbff5a8 (num의 주소와 동일)
    printf("포인터 p가 가리키는 값: %d\n", *p); // 10 (*p는 num과 동일)

    *p = 20; // 포인터가 가리키는 주소의 값을 20으로 변경
    printf("변경된 num의 값: %d\n", num);     // 20

    return 0;
}

C 언어의 꽃이자 정보처리기사 실기의 핵심인 '포인터'의 기본 개념을 정리합니다. 포인터는 변수의 값이 아닌 **'메모리 주소'**를 저장하는 특별한 변수입니다.

포인터 변수 선언 (*): int *p; 형태로 선언하며, 이는 정수형 데이터가 저장된 메모리 주소를 담겠다는 의미입니다.

주소 연산자 (&): 일반 변수 앞에 붙여서 해당 변수가 메모리의 어디에 위치해 있는지 그 '주소값'을 반환합니다. (예: p = &num;)

간접 참조 연산자 (*): 포인터 변수 앞에 붙여서 해당 주소로 찾아가 그 안에 들어있는 '알맹이(값)'를 꺼내거나 수정합니다. (예: *p = 20;)
간단히 말해, &는 집 주소를 알아내는 것이고, *는 그 주소로 직접 배달을 가는 것과 같습니다.
