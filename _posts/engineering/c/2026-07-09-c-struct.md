C언어에서 배열과 구조체는 여러 개의 데이터를 한곳에 모아두는 '상자' 역할을 합니다. 하지만 내용물에서 결정적인 차이가 있습니다.

* **배열 (계란판):** 같은 타입의 데이터만 모을 수 있습니다. (`int`만 10개, `char`만 5개)
* **구조체 (종합 도시락):** 서로 다른 타입의 데이터를 한곳에 모을 수 있습니다. (`이름(char)`, `나이(int)`, `시력(float)`을 하나의 세트로 묶음)

## 1. 구조체 기본: 점(`.`) 연산자

구조체는 먼저 '설계도(붕어빵 틀)'를 만들고, 그 설계도를 바탕으로 '변수(실제 붕어빵)'를 찍어내서 사용합니다.
구조체 안의 알맹이(멤버)를 꺼낼 때는 **점(`.`)**을 찍습니다.

```c
#include <stdio.h>

// 1. 설계도 만들기 (Student라는 새로운 타입 생성)
struct Student {
    char name[10];
    int age;
};

int main() {
    // 2. 변수 만들기 (설계도를 바탕으로 s1이라는 변수 생성 및 초기화)
    struct Student s1 = {"홍길동", 20};

    // 3. 일반 구조체 변수는 점(.)을 찍어서 알맹이에 접근합니다.
    printf("이름: %s\n", s1.name);
    printf("나이: %d\n", s1.age);

    return 0;
}
```

## 구조체 포인터와 화살표(`->`) 연산자

일반 구조체 변수에는 점(`.`)을 찍지만, 구조체의 주소를 가리키는 '포인터'가 등장하면 화살표(`->`)를 써야 합니다.
우리가 앞서 포인터를 배울 때, 주소로 찾아가서 알맹이를 꺼내려면 `*`를 쓴다고 배웠습니다. 구조체도 마찬가지입니다.

```c
#include <stdio.h>

struct Student {
    char name[10];
    int age;
};

int main() {
    struct Student s1 = {"홍길동", 20};
    
    // s1의 주소를 담는 구조체 포인터 p 선언
    struct Student *p = &s1;

    // 방법 A: 정석대로 껍데기 까기 (괄호 필수)
    // p가 가진 주소로 찾아가서(*), 그 안의 age를 꺼내라(.)
    printf("나이(정석): %d\n", (*p).age); 

    // 방법 B: 화살표(->) 단축키 사용 (정처기 단골 출제)
    // C언어 개발자들이 (*p).age 라고 치기 귀찮아서 만든 단축키입니다.
    printf("나이(단축키): %d\n", p->age);

    return 0;
}
```
